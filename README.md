# 🚀 Landing Page - AlejoDev

Landing page moderna y responsive desarrollada con HTML, CSS y JavaScript.

## 🛠️ Tecnologías
- HTML5
- CSS3 (Flexbox, variables, responsive)
- JavaScript (DOM, eventos, validación)
- Intersection Observer API

## ✨ Funcionalidades
- Menú hamburguesa responsive
- Animaciones al hacer scroll
- Hover effects en cards
- Formulario con validación

## 📱 Responsive
Adaptado a dispositivos móviles y desktop.
